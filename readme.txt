== Description ==

If you want to build an online WordPress business directory listing then you are in the right place. Clasify Classified Listing is the powerful WordPress business directory listing plugin that will help to create a professional directory listing. 


This WordPress business directory listing plugin is easy to use. It allows you to create and manage a directory of businesses and individuals. The plugin includes a listing of all businesses in your directory, classified ads, booking, local directory, service directory, and any kind of listing for each business. This plugin is fully customizable and has rich features to create your listing site wholesomely.

== Installation ==

This section describes how to install the plugin and get it working.
 
e.g.
 
1. Upload `clasify-classified-listing` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start using it by editng Clasify Classified menu in the dashboard
